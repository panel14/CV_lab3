Реализация алгоритма классификации изображений на основе сверточной нейронной сети (CNN)

## Теоритическая база

Сверточные нейронные сети (Convolutional Neural Networks, CNN) - класс нейросетевых архитектур, который активно используется в задачах классификации изображений, обнаружении объектов и др.

В случае классификации таки сети с помощью слоёв свертки извлекают локальные признаки изображения (линии, углы, вертикальные края, размытые объекты), которые затем сжимаются (слои Pooling) и на их основе полносвязные слои (Dense) делают финальные предсказания классов.

# Описание разработанной системы

## Датасет
В ходе работы был использован [датасет](https://www.kaggle.com/datasets/antobenedetti/animals) для классификации животных, включающий 5 классов:

- Коты
- Собаки
- Львы
- Слоны
- Лошади

Датасет включает в себя данные для обучения (train) и данные для валидации (val).

Тестовые данные собирались вручную, по 10 изображений на каждый класс.

Тестовые данные включают в себя как стандартные изображения (крупный план, животное видно полностью либо только морда) так и не стандартные (животное видно не полностью / несколько животных одного класса на изображении).

## Архитектура CNN

Сверточная сеть включает в себя:
- Слой аугментации данных (разворот ~ 10%, зум ~ 10%, поворот по горизонтальной оси)
- 4 сверточных слоя
- Слои классификации

Количество эпох обучения модели - 20
